# Crosslinguistic Influence on Subject Pronouns in Bilingual Spanish-English Children

## Installation Instructions

Steps to run the code as well as commands for install are outlined below:

1. Download the repository
2. Install the requirements using the Python kernel for your Jupyter notebooks
3. Update the filepath variables in "Project401v2.ipynb" and "finalPres.ipynb" to point to the data in the repository
4. Run the code

Note: If your Jupyter kernel uses a python kernel other than the system default, include "(kernel path) -m" before the pip install command and replace "python" in the spacy command with the kernel path

```
git clone https://github.com/Kennedy12202/LING401Project.git
pip install -r requirements.txt
python -m spacy download "en_core_web_sm"
```
